## 🛡️ Project Overview

Modern aviation relies heavily on digital communication (ADS-B, ARINC 429, ACARS), which were often designed before the era of advanced cyber warfare. This repository enables:

1.  **Detection of Ghost Aircraft**: Using ML to flag spoofed ADS-B signals.
2.  **Avionics Bus Protection**: Identifying malicious command injections in flight control systems.

With initiatives like the **FAB's Aerospace Cyber Defense Center (CDCAER)** and **Embraer's** growing focus on cyber-resilience, the need for AI-driven security in aviation is critical in 2026.

## 🚀 Key Modules (TRL-9 Architecture)

### 1. ADS-B Spoofing & MLAT-Correlation (`src/adsb_spoofing.py`)
**Framework**: **DO-260B (1090 MHz Extended Squitter)**
**Outcome**: Neutralizes "Impossible Physics" tracks, validating physical layer integrity, producing an immutable audit trail for legal compliance.

### 2. Edge-to-Cloud Vertex AI / Gemini Integration (`integration_edge_vertex.py`)
**Framework**: **Google Cloud (BigQuery, Vertex AI, ARIMA_PLUS)**
**Functionality**: Secures Edge AI Nodes routing to Google Cloud. Only LGPD-compliant SHA-256 forensic hashes and anonymous metric payloads are transmitted via Gemini API.

### 3. Hardware-in-The-Loop (HITL) SDR Node (`sdr_hitl_edge_node.py`)
**Framework**: **Software Defined Radio (RTL-SDR / dump1090)**
**Functionality**: Elevates the project to TRL-7 Operational Environment. Integrates physical USB radio receivers capturing live 1090MHz Baseband ADS-B signals in São Paulo airspace.

### 4. TRL-9 Production Operational Auditor e Cloud Run (`src/agent_cloud_engine.py`)
**Framework**: **Python Serverless / ISO-27001 Forensic Chain / Google Cloud Run**
**Functionality**: O **Motor Web de Inteligência Artificial** para a defesa do espaço aéreo. Hospedado publicamente no Google Cloud Platform configurado para zero downtime e zero-cost em espera (request-based). O projeto processa JSONs de SDR da borda diretamente para a Vertex AI TRL-9 em latências sub-200ms. O Cloud Build atua como pipeline CI/CD via GitHub.
