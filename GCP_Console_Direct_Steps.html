GCP Console Setup: We Can Fly V2

Guia super facilitado com as 4 etapas cruciais para conectar a nuvem ao seu terminal.

1
🌐 Criar o Novo Projeto no Site

1. No site do Google Cloud Console, vá no canto superior esquerdo e clique no botão de selecionar um projeto.

2. Clique em Novo Projeto (New Project).

3. Sugestão de ID de Projeto para você usar:

ita-wecanfly-v2-dev

Atenção: Mantenha este ID anotado. Ele é intransferível e único.

2
💳 Ativar o Faturamento (Sem custo)

1. Assim que o projeto ita-wecanfly-v2-dev for criado e você entrar nele, procure o menu lateral esquerdo.

2. Clique em Faturamento (Billing).

3. Clique em Vincular conta de faturamento e associe a conta que possui os $300 dólares (ou o Trial de 30 dias do Gemini Enterprise).
Isso garante que o projeto saia da restrição de "sandbox".

3
🧠 Ativar as APIs de Inteligência Artificial

1. Na barra de pesquisa superior (Search), digite Vertex AI.

2. A tela vai pedir para Ativar TODAS as APIs recomendadas (Enable all recommended APIs).

3. Clique no botão azul Ativar (Enable). Aguarde uns segundos até carregar o painel Studio.

4
⏬ Instalador do Google Cloud SDK

O seu Windows retornou o erro de que o "gcloud não é reconhecido". Para resolvermos isso trabalhando juntos, eu acabei de pedir ao seu terminal para baixar e abrir o instalador oficial!

1. Volte para a tela do seu computador e vá apertando Avançar (Next) e Instalar (Install) no processo de instalação que abriu.

2. Assim que concluir, para que o Windows consiga enxergar a ferramenta recém-instalada, você deve obrigatoriamente FECHAR SEU TERMINAL antigo.

3. Feito isso, abra uma janela de terminal nova na sua pasta atual.

5
💻 Sincronizar o seu Windows com a Nuvem

No terminal NOVO, agora você pode disparar os comandos sem medo do erro antigo:

1. Faça o Login com o seu E-mail:

gcloud auth application-default login

2. Aponte o Windows para o nosso ID de projeto recém criado:

gcloud config set project ita-wecanfly-v2-dev
✔️ Pronto! Assim que fizer isso, nossa V2.0 e qualquer inteligência ADK já podem rodar no seu computador sugando de graça o processamento poderoso do Vertex AI do Google!
