Lab Walkthrough: Get Started with ADK

Direct commands to finish the lab quickly!

💻 Task 1 & 3: Environment Setup & Search Agent

1. Open the Cloud Shell Terminal in the lab console.

2. Copy & run this to get the code (Replace YOUR_GCP_PROJECT_ID with your unique lab ID shown in the left panel):

gcloud storage cp -r gs://YOUR_GCP_PROJECT_ID-bucket/* .

3. Install the Agent Development Kit (ADK):

export PATH=$PATH:"/home/${USER}/.local/bin"
python3 -m pip install google-adk -r adk_project/requirements.txt

4. Open Cloud Shell Editor, navigate to adk_project/my_google_search_agent/agent.py, and add this line at the bottom inside the Agent creation parenthesis (don't forget the comma on the previous line if needed!):

tools=[google_search]
🌐 Task 4: Run the ADK Web UI

1. In the terminal, run this block to set the environment (Remember to swap YOUR_GCP_PROJECT_ID!):

cd ~/adk_project
cat << EOF > my_google_search_agent/.env
GOOGLE_GENAI_USE_VERTEXAI=TRUE
GOOGLE_CLOUD_PROJECT=YOUR_GCP_PROJECT_ID
GOOGLE_CLOUD_LOCATION=global
MODEL=gemini_flash_model_id
EOF

2. Start the local server:

adk web

3. Click the http://127.0.0.1:8000 link in the terminal to open the UI. Ask the agent: What is some recent global news?

4. Close the browser tab, go back to terminal, and hit CTRL+C to stop.

✅ Click "Check my progress" for Task 4.
⚙️ Task 5: Run Programmatically & Add Schema

1. Run these to export variables (Again: YOUR_GCP_PROJECT_ID):

export GOOGLE_GENAI_USE_VERTEXAI=TRUE
export GOOGLE_CLOUD_PROJECT=YOUR_GCP_PROJECT_ID
export GOOGLE_CLOUD_LOCATION=global
export MODEL=gemini_flash_model_id

2. Open adk_project/app_agent/agent.py and paste this right after the other imports at the top:

from pydantic import BaseModel, Field

class CountryCapital(BaseModel):
    capital: str = Field(description="A country's capital.")

3. In the root_agent definition inside the same file, add this parameter:

output_schema=CountryCapital,

4. Run the code:

python3 app_agent/agent.py
✅ Click "Check my progress" for Task 5.
🗣️ Task 6: CLI Chat Interface

1. Run this in the terminal:

adk run my_google_search_agent

2. Ask it anything, like: what are some new movies that have been released in the past month in India?

3. Type exit to stop.

✅ Click "Check my progress" for Task 6.
⚖️ Task 7: Multi-Agent Auditor Example

1. Set up the environment for the multi-agent auditor:

cd ~/adk_project
cat << EOF > llm_auditor/.env
GOOGLE_GENAI_USE_VERTEXAI=TRUE
GOOGLE_CLOUD_PROJECT=YOUR_GCP_PROJECT_ID
GOOGLE_CLOUD_LOCATION=global
MODEL=gemini_flash_model_id
EOF

adk web

2. Open the web link again. Select the llm_auditor agent on the left.

3. Give it a false statement to audit: Double check this: Earth is further away from the Sun than Mars.

4. Close the tab, hit CTRL+C and you are entirely done with the lab!

✅ Click "Check my progress" for Task 7.
